# 定制返回结果字段

// 多个字段用逗号分隔，就能返回指定的字段了

GET /test_index/test_type/1?_source=test_content2,test_content
1
2
3

dsl 语法，只有一个字段的时候，可以直接写 "_source": "test_content"

GET /test_index/test_type/_search
{
  "query": {
    "match": {
      "_id": "1"
    }
  },
  "_source": ["test_content","test_content2"]
